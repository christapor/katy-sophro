# 🌿 Katy Sophro

Site vitrine pour **Katy Lengrand**, sophrologue certifiée RNCP basée à Boisset (Haute-Loire).

---

## 🎯 Objectif du projet

Ce site a pour but de :

* Présenter les prestations de sophrologie
* Améliorer le référencement local (SEO)
* Générer des prises de contact (téléphone / email)
* Offrir une vitrine professionnelle simple, rapide et efficace

---

## 🧰 Technologies utilisées

Projet volontairement simple et léger :

* **HTML5** (structure du site)
* **CSS3** (design et responsive)
* **Google Fonts** (typographies)
* **Schema.org (JSON-LD)** pour le référencement local

👉 Aucun framework / build tool (site statique)

---

## 🚀 Lancer le projet

Aucune installation nécessaire.

### Option 1 :

Ouvrir directement le fichier :

```
index.html
```

### Option 2 (recommandé pour dev) :

Utiliser un serveur local (ex : VS Code + Live Server)

## 🎨 Design & UX

* Design orienté **bien-être / sérénité**
* Palette de couleurs douce (bleu / vert / accent doré)
* Typographies élégantes (Playfair Display + Montserrat)
* Responsive mobile optimisé
* Navigation simple (1 page)

## 🔍 SEO (Référencement)

Optimisations mises en place :

* Balises meta (title + description)
* Mots-clés locaux
* Structure HTML sémantique
* Données structurées **Schema.org (HealthAndBeautyBusiness)**
* Optimisation pour les recherches locales :

  * Boisset
  * Retournac
  * Craponne-sur-Arzon
  * Usson-en-Forez
